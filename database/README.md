# Database

Folder ini berisi script SQL untuk migrasi dan stored procedure.

## Struktur Folder

- `migrations/` — DDL untuk membuat tabel baru (jalan sekali di SSMS)
- `stored-procedures/` — Stored procedure pengganti row SQL di API route

## Cara Pakai

1. Buka SSMS, konek ke server yang sesuai
2. Buka file `.sql` yang diinginkan
3. Execute (F5)
