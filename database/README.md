# Database

Folder ini berisi script SQL untuk migrasi dan stored procedure.

## Struktur Folder

- `sp_full.sql` — Kumpulan semua stored procedure
- `schema_full.sql` — Full CREATE TABLE schema snapshot


## Cara Pakai

1. Buka SSMS, konek ke server yang sesuai
2. Buka file `.sql` yang diinginkan
3. Execute (F5)
